How to Use

Launch the application by running the gui class.

Enter the desired array size in the "Enter Array Size" field.

Click "Set Size" to generate input fields for each array element.

Enter integers into each field.

Click "Show Array" to visualize the array along with index labels.

If invalid input is entered, an error dialog will prompt the user.

Requirements

Java Development Kit (JDK) 8 or higher.

No additional libraries required.

Notes

The array is limited to integer values.

The display panel uses color coding (cyan) to differentiate array values.
